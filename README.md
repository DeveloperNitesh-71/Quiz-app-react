# React Quiz App

An interactive Quiz Application built using **React.js** that allows users to answer multiple-choice questions, submit their responses, view their final score, and restart the quiz.

This project was created to practice React fundamentals including state management, event handling, and conditional rendering.

---

## 🚀 Features

- 📝 20 quiz questions
- ✅ Multiple-choice answers
- 📊 Automatic score calculation
- 🎯 Result screen after submission
- 🔄 Restart quiz functionality
- ⚡ Dynamic UI updates
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

## 🎮 How It Works

1. Start the quiz.
2. Answer all 20 questions.
3. Click the **Submit** button.
4. View your final score.
5. Click **Restart Quiz** to start again.
